# Sila Web - SvelteKit app

A thin SvelteKit wrapper around `@sila/client`.

This package hosts the web version of the frontend, while shared UI and client logic live in `packages/client`.

## Dev

From the repo root:

```sh
npm run dev -w web
```

Or from this package:

```sh
npm run dev
```

## Build

```sh
npm run build
```
