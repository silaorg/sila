# Sila Workbench

This package hosts a SvelteKit website used to:

- Show components of Sila
- Test components with Playwright (UI integration tests)
- Provide a workbench to develop components in isolation from the main app

## Development

- From repo root: `npm run workbench`
- Directly from package: `npm run dev -w packages/workbench`

### Breakpoints

If you want to use breakpoints in VSCode / Cursor - run "Debug Workbench" from the Debug section of the editor.
