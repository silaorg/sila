# Silaland App - SvelteKit

A thin SvelteKit app for the hosted Silaland web interface.

Right now this package is a simple starter with Skeleton wired in and a minimal page shell. Over time it will become the browser app for end users and admins, backed by the land runtime and Neorest API.

## Dev

From the repo root:

```sh
npm run dev -w app
```

Or from this package:

```sh
npm run dev
```

## Build

```sh
npm run build
```
