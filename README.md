# Sila2

Sila is AI that does things for teams and businesses. Sila is hosted on Linux servers with AI agents that can use CLI and different tools to help with tasks - analyzing data, checking logs, reviewing code, support tickets and more. Users communicate with the agents from different channels including Slack and Telegram. It's possible to setup any channel from chats to phone calls and email.

Sila2 is a new version of Sila, made from scrach in this branch (v2). v2 branch was created as an orhpan branch without the commit history of v1.

You can quickly access sila v1 repo here: ../sila and a prototype of v2 in ../telegram-bot
